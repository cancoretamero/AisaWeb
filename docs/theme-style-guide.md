# Theme Style Guide

## Light palette (from `/light-mode/`)
- Background: `#FAFAFA`
- Text: `#1F2937`
- Surface/Panel: `#FFFFFF` / `#F3F4F6`

## Dark palette (from original root pages)
- Background: `#050505` or `#020202` (varies by page)
- Text: `#E5E7EB` / `#D1D5DB` (match original per page)

## Glass treatments
- `.glass-apple` (light): translucent white, soft black border, gentle shadow.
- `.glass-apple` (dark): translucent dark, soft white border, deep shadow.

## Patterns
- Grid pattern light: black with low opacity.
- Grid pattern dark: white with low opacity.

## Navbar rules
- Light: invert logo (`filter brightness-0`), links `text-gray-600/700` with `hover:bg-black/5`.
- Dark: logo default, links `text-gray-200/300` with `hover:bg-white/10`.
